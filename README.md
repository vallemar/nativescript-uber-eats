Clone of the main views Uber Eats application. Made with NativeScript-Vue3.

[Playground Preview](https://stackblitz.com/edit/nativescript-vue-uber-eats?file=src%2Fapp.ts&title=NativeScript%20Starter%20Vue) ⚡


Note: Keep in mind that this clone uses many images, these are downloaded from free APIs so the first load will be slow in displaying the images (the slowness comes from the image servers, NOT the app), once downloaded it will They are cached for the next viewing and will be displayed instantly ⚡.
